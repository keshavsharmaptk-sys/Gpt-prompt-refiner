# GPT Prompt Refinement & Evaluation Tool

A frontend-only tool built using HTML, CSS, and JavaScript that uses GPT to:
- Refine prompts
- Detect ambiguity
- Score prompt quality
- Evaluate LLM outputs

## Demo
This project can be run by opening `index.html` in a browser.

## Setup
1. Clone the repo or download files
2. Open `script.js`
3. Add your OpenAI API key:
   const API_KEY = "YOUR_OPENAI_API_KEY";
4. Open `index.html`

## Notes
- This is a demo/portfolio project.
- API key exposure is acceptable for demo purposes only.

## Use Case
Simulates annotation-style prompt refinement and evaluation workflows
similar to AI data quality roles.
